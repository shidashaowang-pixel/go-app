# 需求文档

## 1. 应用概述

- **应用名称**：少儿围棋学习平台
- **应用描述**：面向初学围棋的儿童及其家长、围棋教学者的综合性围棋学习与对弈平台，注重易用性、社交性与健康益智，涵盖围棋文化启蒙、基础入门、题目练习、真人视频图文课程及人机/真人对弈等核心功能。

---

## 2. 用户与使用场景

### 2.1 目标用户
- 儿童（初学围棋，主要使用者）
- 儿童家长（监督、陪伴学习）
- 围棋教学者（发布课程、管理学员）

### 2.2 核心使用场景
- 儿童通过文化启蒙、基础入门、闯关练习逐步学习围棋
- 儿童与AI或真人进行对弈练习
- 教学者发布视频/图文课程供儿童学习
- 家长查看孩子学习进度与对弈记录

---

## 3. 页面结构与功能说明

### 3.1 整体页面结构

```

├── 登录/注册页
├── 首页（导航入口）

│   │   ├── 文化启蒙
│   │   ├── 趣味故事

│   ├── 基础入门（AI教学）
│   ├── 基础定式闯关
│   │   ├── 第1关
│   │   ├── 第2关
│   │   └── 第3关
│   ├── 题目练习
│   └── 真人教学（视频/图文课程）
├── 对弈中心
│   ├── 人机对弈
│   └── 真人对弈

    └── 学员管理
```

### 3.2 登录/注册页
- 支持微信授权登录与账号密码登录两种方式
- 注册时选择角色：儿童、家长、围棋教学者
- 不同角色注册后进入对应功能视图
- 儿童账号可由家长代为注册并绑定家长账号

### 3.3 首页
- 展示平台核心功能入口：学习中心、对弈中心、社交与排行、个人中心
- 根据登录角色展示差异化内容（儿童/家长/教学者）
- 展示学习进度概览与最近对弈记录

### 3.4 学习中心

#### 3.4.1 围棋文化
- **文化启蒙**：图文/视频形式介绍围棋是什么、起源历史、人们喜爱围棋的原因
- **趣味故事**：以儿童易于理解的方式呈现围棋相关趣味故事，图文结合
- **经典案例**：收录玄玄棋经、三十六计、围棋宝典等经典内容，以图文形式展示

#### 3.4.2 基础入门（AI教学）
- AI引导式教学，逐步讲解围棋基础规则（落子、提子、气、胜负判定等）
- 每个知识点配有互动演示，儿童可在棋盘上跟随AI操作练习
- 学习进度自动记录，支持断点续学

#### 3.4.3 基础定式闯关
- 共设3关，难度递进
- 每关包含若干定式题目，儿童需在棋盘上完成正确落子
- 通关后解锁下一关，进度持久化保存
- 闯关结果给予即时反馈（通过/未通过提示）

#### 3.4.4 题目练习
- 提供围棋练习题库（死活题、手筋题等基础题型）
- 儿童在棋盘上作答，系统给出正误反馈
- 练习记录（答题数、正确率）持久化保存

#### 3.4.5 真人教学（视频/图文课程）
- 教学者发布的视频课程与图文课程均在此展示
- 儿童可按课程列表浏览、选择并学习
- 支持视频播放与图文阅读
- 学习进度（已学/未学）持久化保存

### 3.5 对弈中心

#### 3.5.1 人机对弈
- 提供三个难度档位：初级、初级AI、AI（大模型算力支持）
- 儿童选择难度后进入棋盘对弈界面
- 支持悔棋、认输操作
- 对弈结束后展示胜负结果，战绩持久化保存

#### 3.5.2 真人对弈
- 支持好友对战与随机匹配两种模式
- 对弈界面与人机对弈一致，支持悔棋申请（需对方同意）、认输
- 对弈结束后展示胜负结果，战绩持久化保存

### 3.6 社交与排行

#### 3.6.1 好友列表
- 支持搜索用户并添加好友
- 好友列表展示好友在线状态
- 可向好友发起对弈邀请

#### 3.6.2 排行榜
- 展示平台内儿童用户的对弈积分排名
- 支持按周/月/总榜切换

#### 3.6.3 成就分享
- 儿童完成闯关、练习达标等里程碑时获得成就徽章
- 支持将成就分享至微信

### 3.7 个人中心
- **学习进度**：展示围棋文化、基础入门、闯关、题目练习、课程学习的完成情况
- **对弈战绩**：展示人机对弈与真人对弈的历史记录（胜/负/平、对手、时间）
- **账号设置**：修改昵称、头像、密码；家长账号可查看绑定儿童账号的学习与对弈数据

### 3.8 教学者后台（仅教学者角色）

#### 3.8.1 课程管理
- 新建、编辑、删除课程
- 支持上传视频文件与图文内容
- 课程发布后在真人教学模块对儿童可见

#### 3.8.2 学员管理
- 查看学员列表及其学习进度数据

## 4. 业务规则与逻辑

1. **角色权限**：儿童账号可访问学习中心、对弈中心、社交与排行、个人中心；家长账号额外可查看绑定儿童的学习与对弈数据；教学者账号额外可访问教学者后台。
2. **闯关解锁规则**：第1关通过后解锁第2关，第2关通过后解锁第3关，不可跨关进入。
3. **对弈积分规则**：真人对弈胜利获得积分，负局扣除积分，积分用于排行榜排名；人机对弈不计入排行榜积分。
4. **好友对战邀请**：发起邀请后，对方需在规定时间内接受，超时自动取消。
5. **悔棋规则（真人对弈）**：申请悔棋需对方同意方可生效，每局悔棋次数不限但需对方每次确认。
6. **数据持久化**：闯关进度、题目练习记录、课程学习进度、对弈战绩均通过后端数据库持久化存储，用户登录后可继续上次进度。
7. **微信分享**：成就分享调用微信分享接口，需用户授权。

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 账号密码登录时密码错误 | 提示「账号或密码错误，请重试」，连续5次错误后锁定账号10分钟 |
| 真人对弈对方掉线 | 等待60秒后判定掉线方负局，胜方正常获得积分 |
| 人机对弈AI服务不可用 | 提示「AI服务暂时不可用，请稍后再试」，不进入对弈 |
| 视频课程加载失败 | 提示「视频加载失败，请检查网络后重试」 |
| 闯关答题超时（如设有时限） | 本关判定未通过，提示重新挑战 |
| 好友邀请超时未响应 | 自动取消邀请，提示发起方「对方未响应，邀请已取消」 |
| 教学者上传课程文件过大 | 提示文件大小限制，要求重新上传 |

## 6. 验收标准

1. 微信授权登录与账号密码登录均可正常完成，三种角色注册后进入对应功能视图。
2. 围棋文化三个子模块（文化启蒙、趣味故事、经典案例）内容可正常展示。
3. 基础入门AI教学可正常引导儿童完成互动练习，进度正确保存。
4. 基础定式闯关三关可正常进行，通关后正确解锁下一关，进度持久化。
5. 题目练习可正常作答并给出正误反馈，练习记录正确保存。
6. 真人教学课程列表正常展示，视频播放与图文阅读功能正常，学习进度正确保存。
7. 人机对弈三个难度档位均可正常进行，战绩正确保存。
8. 真人对弈好友对战与随机匹配均可正常进行，悔棋申请流程正确，战绩正确保存。
9. 好友添加、排行榜展示、成就徽章获取与微信分享功能正常。
10. 个人中心学习进度与对弈战绩数据展示准确。
11. 教学者后台可正常发布、编辑、删除课程，学员数据可正常查看。
12. 家长账号可正常查看绑定儿童的学习与对弈数据。
